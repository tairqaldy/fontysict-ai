## ğŸ’¡ Pro Design Tips

#### Start with User Stories

"As a [user], I want to [goal], so that [benefit]." Design the wireframe around completing this story.

#### Mobile First Always

Design for the smallest screen first. If it works on mobile, desktop is easy. Not the other way around.

#### Test Early & Often

Show your wireframes to real users. Watch where they get confused. Fix it before you code.
